Symbolic Pattern Matching

  1. Create a symbol from the current features.
  2. Add the current symbol to a stored sequence.
  3. Search the sequence for the current symbol.
  4. If found, move to the symbol adjacent to the matched symbol and check for another match.
  5. Record the longest match. This is most like the current sequence and may lead to a similar outcome.

Why Symbols?

  • Discrete value
  • Limited alphabet
  • Existing algorithms

Symbolic Pattern Matching

Advantages
  • Fairly straightforward
  • No math
  • Existing algorithms, especially from string processing
  • Long match implies strong prediction

Symbolic Pattern Matching

Disadvantages
  • Space and Time as n grows
  • Doesn't exactly generalize
  • Small alphabet means coarse measurement
  • Relatively slow learning

Clustering / Nearest Neighbor Search

  1. Search for k observations most similar to the current observation.
  2. Determine the most likely outcome from the matched observations.
  3. When the outcome is known, store it.

Artificial Neural Networks

  1. Feed features into the impenetrable network.
  2. Receive a projected outcome and hope it's correct.
  3. When the outcome is known, adjust the network.
Cost function - usually sigmoid: 
(2 / (1 + Math.exp(-2 * x))) - 1;

Backpropagation

Use the derivative of the cost function to adjust weights.

ANN

Advantages
  • Small representation that doesn't grow
  • Good generalization
  • Fast learning with enough tweaks

ANN

Disadvantages
  • Impenetrable
  • Temperamental
  • Over fitting
  • Local minima
