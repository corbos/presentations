SVG objects are members of the DOM.

(they're scriptable)

And they can be styled using CSS

Standards

  • SVG is a W3C Recommendation
  • <canvas> is a Candidate Recommendation
  • Google started indexing SVG content in 2010
  • Supported by the most recent versions of all major browsers

SVG Disadvantages

  • Full support is unpredictable - full, tiny, and mobile profiles. (This is eased by some SVG libraries.)
  • Rendering slows down as objects are added to the DOM.
